# recipe-crud

Simple CRUD app. It uses `db.json` file as a fake API. 

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Run json-server

```sh
npx json-server --watch db.json
```
