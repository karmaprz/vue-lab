# vue-lab

## Topics

- Attribute binding
- Conditional rendering
- List rendering
- Event handling
- Components & Props
- Communicating events
- Forms

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
